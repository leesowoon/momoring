# Momoring 로드맵 03 — STS 아키텍처 & MVP 구현

## 목표
사용자 음성을 받아 Momoring이 음성으로 응답하는 최소 동작 파이프라인(STT→LLM→TTS)을 구축합니다.

## 기간
2–3주

## 핵심 작업
1. STS 파이프라인 구현
   - 입력: 사용자 음성
   - 처리: STT, LLM 응답 생성
   - 출력: TTS 음성 재생
2. 지연 시간(레이트턴시) 목표 수립 및 계측
   - 예: 체감상 자연스러운 왕복 응답 시간 목표 설정
3. 프롬프트 정책 연결
   - 연령별 난이도 제어
   - 안전 응답 기본 템플릿 연동
4. 실패 대응 로직
   - STT 실패/무응답/네트워크 지연 시 fallback

## 완료 기준 (DoD)
- 엔드투엔드 음성 대화 MVP 동작
- 핵심 오류 상황 fallback 동작 확인
- 기본 성능 계측 대시보드(또는 로그) 준비
