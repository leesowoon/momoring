# Momoring 로드맵 02 — STS UX & 페르소나 설계

## 목표
Momoring이 화면에 등장해 자연스럽게 듣고 말하는 경험을 구체화하고, 캐릭터/목소리/대화 톤을 제품화 가능한 수준으로 정의합니다.

## 기간
1–2주

## 핵심 작업
1. STS 대화 UX 설계
   - 발화 시작 방식(Push-to-talk/VAD)
   - 끼어들기(barge-in) 허용 정책
   - 답변 길이/속도/턴 교환 규칙
2. Momoring 페르소나 가이드 확정
   - 말투: 다정함, 공감, 비난 금지
   - 구조: 설명 → 예시 → 되물어보기
3. 보이스 가이드 정의
   - 음색, 속도, 감정 표현 레벨
4. 캐릭터 상태 정의
   - 대기/듣는 중/생각 중/말하는 중

## 완료 기준 (DoD)
- UX 플로우 차트 완성
- 페르소나/보이스 가이드 v1 확정
- 캐릭터 상태 전이표 완성
